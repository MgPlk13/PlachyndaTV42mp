# MyMasters2025 — Cyber Defense for DBMS (Flask + PostgreSQL + Analytics + ML)

Веб-застосунок для перевірки SQL-запитів, симуляції навантаження (DoS), логування подій та аналітики, а також
перенавчання ML-моделі на основі логів.

---

## Бази даних

Проєкт працює з двома окремими базами даних, кожна з яких виконує свою функцію
в системі забезпечення кіберзахисту.

### Основна база даних (PostgreSQL)

Використовується для виконання SQL-запитів у режимі тестування.
Через цю базу здійснюється:

- перевірка коректності SQL-запитів,
- симуляція нормальної та шкідливої активності,
- обробка запитів користувача в SQL Tester.

Підключення до бази може виконуватись:

- через файл конфігурації `config_db.py`,
- або динамічно через веб-інтерфейс застосунку.

Тип БД: PostgreSQL (наприклад, навчальна база Pagila).

---

### База аналітики (Analytics, PostgreSQL)

Окрема база даних, призначена для збору та аналізу подій безпеки.

У цій базі зберігаються:

- логи SQL-запитів (дозволені та заблоковані),
- типи атак,
- результати роботи ML-моделі.

База аналітики використовується для:

- побудови статистики та графіків,
- аналізу інцидентів,
- перенавчання ML-моделі на основі накопичених логів.
